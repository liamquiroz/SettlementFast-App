SettlementFast React Native Mobile App - Complete Build Prompt
Project Overview
Build a premium React Native (Expo) mobile app for SettlementFast - a class action settlement discovery platform. The app connects to an existing production backend API and delivers a polished, App Store-ready experience with smooth animations, native feel, and excellent offline capabilities.

Production API Base URL: https://settlementfast.com

Authentication
Use Supabase Auth with the same methods as the web app:

Email/password login
Magic link (passwordless email)
Token Flow:

Authenticate via Supabase Auth SDK (@supabase/supabase-js)
Store tokens securely using expo-secure-store
Include JWT token in Authorization: Bearer <token> header for all API calls
Handle token refresh automatically via Supabase client
Implement automatic session persistence across app restarts
Biometric Authentication:

Implement Face ID / Touch ID using expo-local-authentication
Allow users to enable biometric unlock after initial login
Store encrypted session indicator in secure storage
Auto-prompt for biometric on app resume (configurable)
Fallback to passcode if biometric fails

// POST /api/push-tokens - Register token
// Request:
{
  token: string;                              // Expo push token (required)
  platform: 'ios' | 'android' | 'web';        // Required
  deviceId?: string;                          // Optional device identifier
  deviceName?: string;                        // Optional e.g., "iPhone 15 Pro"
}
// Response: PushToken object
// DELETE /api/push-tokens - Remove specific token
// Request: { token: string }
// Response: 204 No Content
// DELETE /api/push-tokens/all
// Response: 204 No Content
